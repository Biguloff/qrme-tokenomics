Backend implementation

Схема БД, flow списания, API endpoints. Задача для Таймураза.

Схема БД — новые таблицы

-- Баланс монет пользователя CREATE TABLE user_coin_balance ( user_id UUID PRIMARY KEY REFERENCES users(id) ON DELETE CASCADE, balance INTEGER NOT NULL DEFAULT 0 CHECK (balance >= 0), total_purchased INTEGER NOT NULL DEFAULT 0, total_spent INTEGER NOT NULL DEFAULT 0, free_remaining INTEGER NOT NULL DEFAULT 5, updated_at TIMESTAMPTZ NOT NULL DEFAULT NOW() ); -- Пакеты монет (конфигурация) CREATE TABLE coin_packages ( id UUID PRIMARY KEY DEFAULT gen_random_uuid(), name VARCHAR(50) NOT NULL, display_name VARCHAR(100) NOT NULL, coins INTEGER NOT NULL, price_rub INTEGER NOT NULL, price_per_coin NUMERIC(4,2) NOT NULL, discount_percent INTEGER DEFAULT 0, sort_order INTEGER NOT NULL DEFAULT 0, is_active BOOLEAN NOT NULL DEFAULT TRUE, created_at TIMESTAMPTZ NOT NULL DEFAULT NOW() ); -- История покупок CREATE TABLE coin_purchases ( id UUID PRIMARY KEY DEFAULT gen_random_uuid(), user_id UUID NOT NULL REFERENCES users(id), package_id UUID NOT NULL REFERENCES coin_packages(id), coins INTEGER NOT NULL, amount_rub INTEGER NOT NULL, payment_provider VARCHAR(50), payment_id VARCHAR(255), status VARCHAR(20) NOT NULL DEFAULT 'pending', created_at TIMESTAMPTZ NOT NULL DEFAULT NOW() ); -- История списаний CREATE TABLE coin_transactions ( id UUID PRIMARY KEY DEFAULT gen_random_uuid(), user_id UUID NOT NULL REFERENCES users(id), amount INTEGER NOT NULL, balance_after INTEGER NOT NULL, type VARCHAR(20) NOT NULL, model_id VARCHAR(100), bot_username VARCHAR(100), aggai_cost_usd NUMERIC(10,6), metadata JSONB, created_at TIMESTAMPTZ NOT NULL DEFAULT NOW() ); -- Цена генерации для каждой модели CREATE TABLE model_pricing ( id UUID PRIMARY KEY DEFAULT gen_random_uuid(), aggai_model_id VARCHAR(200) NOT NULL UNIQUE, display_name VARCHAR(100) NOT NULL, coins_per_generation INTEGER NOT NULL, cost_usd_estimate NUMERIC(10,6), is_active BOOLEAN NOT NULL DEFAULT TRUE, updated_at TIMESTAMPTZ NOT NULL DEFAULT NOW() );

Flow списания монет

1
Загрузить AiBotConfigEntity → получить aggai_model_id
2
Загрузить model_pricing → получить coins_per_generation
3
Проверить баланс:
• free_remaining > 0 → не списывать, уменьшить free_remaining
• balance >= coins → списать
• balance < coins → отклонить ДО typing indicator
4
Создать coin_transaction (type='generation', amount=-N)
5
Обновить user_coin_balance (balance -= N, total_spent += N)
6
Отправить сообщение в aggai как обычно
7
Записать aggai_cost_usd из ответа в coin_transaction

API Endpoints

GET
/v3/coins/balance
Баланс монет пользователя
→ { balance, free_remaining, total_purchased, total_spent }
GET
/v3/coins/packages
Список пакетов для UI
→ [{ id, name, display_name, coins, price_rub, price_per_coin, discount_percent }]
POST
/v3/coins/purchase
Покупка пакета (после подтверждения платежа)
← { package_id, payment_provider, payment_id } → { purchase_id, coins_added, new_balance }
GET
/v3/coins/transactions?page=1&limit=20
История транзакций
→ { items: [{ id, amount, balance_after, type, model_id, created_at }], total }
GET
/v3/coins/pricing
Цены моделей (стоимость перед генерацией)
→ [{ model_id, display_name, coins_per_generation }]
